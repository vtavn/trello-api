#APIs Trello

## Requirements

* Node 16 or latest
* Git

## Common setup

Clone the repo and install the dependencies.

```bash
git clone https://github.com/vtavn/trello-api.git
cd trello-api
```

```bash
npm install
```

Open Config & rename .env.example to .env

Open [http://localhost:3300](http://localhost:3300) and take a look around.

## Command 
> npm run clean

> npm run build-babel

> npm run build

> npm run production

> npm run dev
